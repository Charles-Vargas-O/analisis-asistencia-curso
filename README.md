# Limpieza y análisis de asistencia a sesiones

Este repositorio contiene el desarrollo de la evaluación técnica para la posición de Analista de Datos.

## Objetivo

Limpiar y consolidar una base de datos que contiene registros de asistencia a cinco sesiones de un curso provenientes de cuatro bases distintas.

## Proceso realizado

- Limpieza y normalización de nombres y correos
- Identificación de participantes únicos
- Consolidación de asistencia a sesiones
- Generación de nombre completo en formato Title Case sin acentos
- Identificación de participantes con menor desempeño
- Análisis exploratorio de asistencia

## Herramientas utilizadas

- R
- tidyverse
- readxl
- stringr
- stringi
- ggplot2

## Resultado

Se generó una base de participantes únicos con su asistencia consolidada y un análisis de la distribución de asistencia al curso.
